# A simple script to automate global exam connection time (With Linux, Windows and Firefox only).

### prerequisite 
1. Python3 
2. Firefox

### Installation
(Do not run this script with sudo)
```
git clone https://github.com/L-e-c-o/GloBotExam.git
pip3 install selenium
cd GloBotExam
python GloBotExam.py

Enter your username
Enter your password
```
Enjoy...
