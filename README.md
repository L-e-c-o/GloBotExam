# GloBotExam
A simple script to automate global exam connection time (with Firefox only).

### Installation
```
git clone https://github.com/L-e-c-o/GloBotExam.git
pip3 install selenium
cd GloBotExam
```
Add the appropriate Firefox driver to your PATH. 
For Linux
```
PATH=$PATH:your/driver/path
```
For Windows with powershell 
```
$Env:Path += ";c:\your\driver\path"
python3 GloBotExam.py
```
